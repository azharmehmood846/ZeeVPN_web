# **App Name**: Zee VPN

## Core Features:

- Landing Page: Displays a hero section with a prominent call to action (CTA) for obtaining a free 6-hour PIN. Also presents a benefits grid, features overview, a 'Fast and Reliable Connection' banner, sections for Premium/Unlimited services, a testimonials carousel, a secondary CTA, and a footer containing the address, social media links, and recent blog posts.
- PIN Generation: Guides users through steps to get, enter, download, and start browsing, along with a button that lets them generate their PIN.
- Educational VPN Article: Presents a detailed educational article explaining what a VPN is. Features a sticky navigation for easy access to different sections of the article.
- App Download Tiles: Provides app platform tiles (Android/iOS/Windows/macOS/TV) with a 'Get Secured Now' call to action for each platform.
- Blog Display: Displays a grid or list of blog posts with thumbnails, date badges, and a 'Read more' link for each post.
- Contact Form: Presents a straightforward contact form with fields for name, phone, email, and message, along with accompanying help text, address, and social media links.
- Free PIN eligibility evaluation: Uses an AI tool to analyze user context to determine if the user is eligible to receive a free PIN or guide them towards premium subscription options.

## Style Guidelines:

- Primary color: Royal Blue (#1E90FF) for a sense of trust and security.
- Background color: Light grayish-blue (##000000).
- Accent color: Emerald Green (#22C55E) for success states, and primary CTA highlights.
- Font: 'Inter' sans-serif for headlines and body, headings at 700, body at 400/500. Note: currently only Google Fonts are supported.
- Sections: Utilize a vertical rhythm of 96-120px with a container max-width of 7xl.
- Cards: Employ rounded-xl corners, a medium shadow, and 24px padding for a modern feel.
- Buttons: Transitions should be solid primary, rounded-lg with hover elevation and a slight scaling effect to show the click.
